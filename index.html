Gestão de Estoque

Vitória Cestas

Acompanhe a saúde do estoque, cadastre produtos e fornecedores e mantenha toda a operação alinhada com o backend já publicado.

Backend pronto (proxy /api para Vercel): https://projeto-vitoriacestas-backend.vercel.app

Fluxo sugerido

  1. Autentique-se e salve o JWT no navegador.
  2. Envie o token em chamadas protegidas (Authorization: Bearer ...).
  3. Liste itens e fornecedores antes de editar ou cadastrar.
  4. Use mensagens da API para mostrar sucesso ou erro.
